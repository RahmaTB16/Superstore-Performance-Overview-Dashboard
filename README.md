Superstore Sales Dashboard – Excel Project
This repository contains an interactive Excel dashboard built using the Superstore dataset.
It provides a complete sales and profitability overview using dynamic pivot tables, slicers, and advanced data visualizations.

1. Project Overview

  The goal of this project is to analyze sales performance across categories, subcategories, and time.
  The dashboard allows users to quickly identify :
  Best-performing categories
  Profit trends
  Loss-making subcategories
  Monthly seasonality
  Correlation between sales & profit
  KPIs of overall business performance
This dashboard is designed for business insights, portfolio presentation, and data analysis practice.

2. Key Features (What the Dashboard Contains)
   
 ✔ KPI Cards
   Total Sales
   Net Profit
   Profit Margin
   Average Order Value (AOV)

✔ Interactive Slicers
   Category
   Sub-Category
   Month   

✔ Visualizations
   Total Sales per Category (Bar Chart)
   Sales Trend Over Time (Line Chart)
   Total Sales & Profit by Sub-Category (Bar Chart with Profit overlay)
   Loss-Making Categories (Bar Chart + notes)
   Sales vs Profit Scatter Plot (Correlation)
   Profit & Sales Pivot Table
   Category Comparison Pivot Table

4. Dashboard Insights (Summary of Findings)

    Office Supplies has the highest total sales.
    Furniture shows negative profit, making it a loss-making category.
    Sales peak around January to March, then decline.
    Accessories and Machines dominate sub-category sales.
    No strong correlation between total sales and profitability.
